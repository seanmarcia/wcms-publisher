WCMS Publisher
=================

Web Content Management System (WCMS) Publisher is the system for managing publishable data for Biola's websites. These include but are not limited to:
- [biola-edu](https://www.biola.edu)
- [web-directory](https://www.biola.edu/directory)
- [student-support](https://studenthub.biola.edu/)
- [blogs](https://www.biola.edu/blogs)
- multi-site: which serves up multiple sites.

##Events
### From Chapel
Chapel events are pulled in using a scheduled sidekiq worker.
